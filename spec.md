# Specification

## Summary
**Goal:** Add bank account linking functionality to allow users to store and manage their bank account number in their profile.

**Planned changes:**
- Add optional bank account number field to user profiles in backend
- Create backend functions to set and retrieve linked bank account numbers
- Build a dedicated bank linking page with a form to input account numbers
- Create React Query hooks for bank account operations
- Add navigation from profile page to bank linking page
- Add route for the bank linking page

**User-visible outcome:** Users can navigate to a bank linking page from their profile, enter their bank account number, and see their linked account displayed on the page and profile.
